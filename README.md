#  Farmer AI Advisory System

An AI-based web application that provides smart farming guidance to farmers.

##  Problem Statement
Farmers often lack instant, reliable and language-friendly advisory support for crop-related decisions.

##  Solution
Our system provides:
- AI-based farming advisory
- Multilingual support (English, Hindi, Marathi)
- Voice-based farmer queries
- Simple and farmer-friendly UI

##  Features
- Text & voice input support
- Multilingual interface
- Predefined advisory questions (MVP)
- Instant AI response panel
